# Selah - Faith & Belief Prayer Generator

A personalized prayer generator website that creates meaningful prayers based on user identity, beliefs, and intentions.

## Features

- Personalized prayer generation
- Support for various faith traditions
- Audio output with voice selection
- Mobile-first design
- Ethical and compassionate content

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Run the development server:
   ```bash
   pnpm dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
pnpm build
```

## Technologies

- Next.js
- React
- TypeScript
- Tailwind CSS
- Radix UI
